# 飞书多维表格 (Feishu Base) 数据库模板

为了方便管理和自动化处理，建议在飞书多维表格中创建一个名为 **「bz49.estt.online 业务管理」** 的表格，并包含以下四个数据表。

---

## 1. 三数占卜 (Oracle)
用于存储三数占卜的请求。

| 字段名称 | 字段类型 | 说明 |
| :--- | :--- | :--- |
| 客户姓名 | 单行文本 | 必填 |
| 性别 | 单选项 | 乾 (男), 坤 (女) |
| 出生日期 | 日期 | 阳历日期 |
| 出生时间 | 单行文本 | HH:mm |
| 出生城市 | 单行文本 | |
| 占卜数字 | 单行文本 | 三个随机数字 |
| 诉求内容 | 多行文本 | |
| 提交时间 | 日期 | 自动生成 |
| 状态 | 单选项 | 待处理, 已回复, 已归档 |

---

## 5. 支付记录 (Payments)

| 字段名称 | 字段类型 | 说明 |
| :--- | :--- | :--- |
| 支付方式 | 单行文本 | 微信支付, 支付宝 |

| 提交时间 | 日期 | 自动生成 |

---

### 导入说明
1. 在飞书多维表格中点击「新建表格」。
2. 根据上述表格手动创建字段。
3. 如果需要自动化同步到数据库，请在环境变量中配置 `FEISHU_APP_ID` 和 `FEISHU_APP_SECRET`。
